✨ Xian Network Telegram Bot ✨ Telegram bot for the Xian project written by Endogen. Xian is a cryptocurrency project and this bot will auto-generate a Xian wallet for you and allow you to send and receive XIAN coins and interact with the network. You can retrieve the private key of your wallet by executing the /address command and then pressing the "Show Private Key" button. IT IS IMPORTANT TO UNDERSTAND THAT THE PRIVATE KEYS OF THE GENERATED WALLETS ARE STORED IN A DATABASE AND CAN BE RECOVERED BY THE DEVELOPER OF THE BOT. STORE ONLY SMALL AMOUNTS OF XIAN IN THE BOT WALLET! If you need a serious and private XIAN wallet, use our Chrome Wallet For a full list of available commands, execute the /help command. Cheers 🍻👋
