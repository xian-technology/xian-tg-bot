✨ Xian Network Telegram Bot ✨ Telegram bot for the Xian project written by Endogen. Xian is a cryptocurrency project and this bot will auto-generate a Xian wallet for you and allow you to send and receive XIAN coins and interact with the network. You can retrieve the private key of your generated wallet by executing the /address command and then pressing the "Show Private Key" button. THE PRIVATE KEY OF THE GENERATED WALLET IS STORED IN A DATABASE AND CAN BE RECOVERED BY THE DEVELOPER OF THE BOT. STORE ONLY SMALL AMOUNTS OF XIAN IN THE BOT WALLET! If you need a serious and private XIAN wallet, use our Chrome Wallet. 💸 We often execute the /bless command in our group to airdrop free tokens to the community. Pre-condition for that is that you have the text "XIAN.ORG" in your Telegram name (NOT username). For a full list of available commands, execute the /help command. Cheers 🍻👋
