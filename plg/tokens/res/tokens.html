Manage your token list so that you are able to send / tip tokens and see their balance. This will not add or remove any tokens in your wallet. It will only manage a custom token contract list. ◾️ Add token: /{{handle}} add con_some_token ◾️ Add all available tokens: /{{handle}} add-all ◾️ Remove token: /{{handle}} remove con_some_token ◾️ Remove all tokens: /{{handle}} remove-all ◾️ List tokens: /{{handle}} list ◾️ Change number of decimal places for a token contract in the /balance command: /{{handle}} decimals con_some_token:2 ◾️ Change ticker for a token contract in the /balance command: /{{handle}} ticker con_some_token:DAMN
